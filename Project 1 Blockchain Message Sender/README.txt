1. Remix IDE used to implement the project virtually.
2. Contracts are deployed on Remix IDE using virtual ETH.

#Blockchain messenger (beginner level project)

The message sent from one account or person to another can be seen by every node(peer) connected to that contract, but the message can only be changed or edited by the original person who sent the message.

Solidity 0.8.15 used.
